Summary: Latitude vs. X

This project analyzed weather factors for over 500 cities collected from the OpenWeatherMap API. The goal was to compare the changes in temperature, cloudiness, humidity and wind speed across different latitudes.

The dataset was cleaned and analyzed using the Matplotlib module for Python, which produced the visualizations provided on this webpage, each with its own explanation and analysis.
